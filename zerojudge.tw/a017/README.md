# a017: 五則運算

## 題目

計算五則運算式的結果，包含加、減、乘、除、餘

## 輸入

輸入資料若干行直到 EOF 為止。每一行包含輸入一個字串，其中包含運算元及運算子，為了方便讀取，所有的運算子及運算元均以空格區隔。  
運算元為 0 ~231 -1 的整數  
運算子則包含 + - \* / % 及 ( )  
運算時請注意先乘除後加減及() 優先運算的計算規則

## 輸出

對每一行輸入，輸出運算結果。  
為了避免小數點誤差，所有的運算過程都不會產生小數點，可以放心使用整數進行運算
