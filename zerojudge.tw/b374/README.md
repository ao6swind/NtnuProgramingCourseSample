# b374: 眾數

## 題目

由文件定義 N 個 1 到 30000 間無序數正整數，其中 1≤N≤10000，同一個正整數可能會出現多次，出現次數最多的雙重稱為眾數。出現的次數。

## 輸入

輸入文件第一行是正整數的個數 N，第二行開始為 N 個正整數。

## 輸出

輸出文件有若干行（因為眾數可能不唯一）  
每行兩個數（之間用 1 個空格隔開），第 1 個是眾數，第 2 個是眾數出現的次數。
