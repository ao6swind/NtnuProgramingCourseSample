# d587: 參貳壹真好吃

## 題目

參貳壹真是太好吃了！

現在有一連串由 1、2、3 這三個數字組成的數列，

請你把他們由小到大排好好嗎？

## 輸入

本題有 2 個測資點，每個 50 分，每個測資點只有一組測資。

第一行有整數 n(1<=n<=1000000)代表接下來的數列有幾個數字

第二行就是這 n 個包含 1、2、3 的數字

## 輸出

對於每組測資，請輸出一行由小到大 1~3 排好的結果。
